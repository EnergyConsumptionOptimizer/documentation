Forecasts

Get all forecasts or filter by utility type

Retrieves energy consumption forecasts. Can be filtered by utility type using query parameter.

Authorizations:
cookieAuth
query Parameters
utilityType
string
Enum: "ELECTRICITY" "GAS" "WATER"

Optional filter by utility type (ELECTRICITY, GAS, WATER)

Responses

Development Server

http://localhost:3000/api/forecasts

Get forecast by specific utility type

Retrieves the forecast for a specific utility type (ELECTRICITY, GAS, or WATER)

Authorizations:
cookieAuth
path Parameters
utilityType
required
string
Enum: "ELECTRICITY" "GAS" "WATER"

The utility type: ELECTRICITY, GAS, or WATER

Responses
